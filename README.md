# GenAI Интенсив — МФТИ 2026

Практический курс по применению GenAI в стартапах — для основателей, которые уже строят продукты.

**Ведущий:** [Егор Рудомёткин](https://t.me/rudometkin_blog)

---

## О курсе

Не "что такое нейросети", а "как прямо сейчас применить в вашем стартапе".

Форматы:
- Разборы кейсов участников в прямом эфире
- Практические воркшопы (пишем код и промпты вместе)
- Антипаттерны от тех, кто уже набил шишки
